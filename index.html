HIRE ME   

CarCat.

A web product built during my final year of undergraduate study, CarCat focuses on the security and validation techniques of both client & server side programming. The site features a consistent UI/UX consideration with Gestalt laws applied.

Space Race

Built at the end of my second year of undergraduate study, Space Race was built for the intention of displaying clear understanding of UI/UX laws. The website focuses on Gestalt Laws along with a clear aesthetic design aimed at educating the younger generation on the space race.

AV1874

AV1874 displays my first experience at designing a website with a clear aspirational design theme along with PHP development, using a data API in order to populate football data ranging from scores to teamsheets. Intense Photoshop development allowed my designing abilities to shine.

Buzzart Studios 2017

At the beginning of my first year of university, I was required to develop a website split into two sections. Firstly the front-end then secondly the back-end. The front-end required the use of a framework for responsiveness and functionality while the back-end required PHP and JS elements in order to create a login, account management and shop system.

Buzzart Studios 2016

Buzzart Studios 2016 was the first website I ever built. Back in College, I was required to create a simple, muiltipage HTML website based on a fictional local business. The website did not require a particular theme introduction, however basic UI desighn such as fonts, colours and content structures were used as conventional formatting.
